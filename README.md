# Анализ фьючерсов на золото (Gold Futures)

**Проект по анализу данных** — курс "Системы искусственного интеллекта"

## Описание проекта

Анализ исторических данных фьючерсов на золото (тикер GC=F) за период 2021–2026 гг.  
Датасет содержит OHLCV цены и 11 технических индикаторов.

### Цели проекта:
- Выполнить первичную обработку и очистку данных
- Провести разведочный анализ (EDA)
- Визуализировать динамику цены и ключевые индикаторы
- Сделать выводы о наборе данных

## Основные выводы

- Цена золота показала **сильный восходящий тренд** за 5 лет (более чем удвоилась).
- Наблюдается повышенная волатильность в 2025–2026 годах.
- Технические индикаторы (MA, RSI, MACD, Bollinger Bands) хорошо отражают рыночные движения.
- Наиболее важные признаки: `close`, `ma_7`, `rsi`, `macd`.
